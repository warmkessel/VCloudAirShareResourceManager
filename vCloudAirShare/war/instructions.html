vCloud Air Share Instructions

Follow the instructions below to begin utilizing a Virtual Machine:

Check out a Machine

To access a virtual machine, we must first reserve it for use by checking it out.

  • On the vCloud Air Share Machines page, log in with your Twitter account.
  • Once you have logged in, you will see a list of virtual machines.
  • Browse to an available virtual machine and click "Checkout"
  • When a dropdown display with an IP address, password, and power button appears, you have successfully checked out the virtual machine.

Establish Remote Connection

This allows you to operate the Virtual Machine from your computer.

  • Click the "Power On" button. A window will pop up confirming that the machine is powered on.
  • Using a command line interface with SSH capabilities, establish a connection with the virtual machine by typing:
    • ssh root@<ip address>
  • Then when prompted, type in the password provided and press enter.
